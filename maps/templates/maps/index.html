If you're new around here, below are a collection of brief summaries detailing what you can expect to see in each tab at the top of the screen:
Map
Searching
  • You will see two search bars at the top left of the screen. The top one will have coordinates in it. IMPORTANT: These coordinates will correspond to your current location IF AND ONLY IF you have "Location Services" turned ON. Otherwise, ignore these coordiantes as they will not be useful to you.
  • At your leisure, type any building names, addresses, or coordinates you so desire into the search bars, and so long as there is a start and end destination, a route will appear on the map taking you where you want to go.
  • You should be able to move the map around and zoom in/out at your convenicne. IMPORTANT: Should you be trying to drag the map around and it doesn't appear to be moving, BEFORE realeasing the mouse, drag it off to the side away from the map! If you don't, the map may accidentally change your route.
  • Just below the map, you can find your current location coordinates (should your "Location Services" be turned on). Also, a simple refresh of the page will put your current location back in the top search bar.
Common Locations
  • Just to the right of the map, you should see a section named "Common Locations". In this section you will see a series of dropdown menus containing many high-traffic areas at UVA. Clicking any of the links in the dropdowns will display a route from your current location to the location described by the link.
  • Since this feature makes use of your current location when displaying routes, please ensure your "Location Services" are on if you plan to use it.
Saved Route
  • If you find yourself searching for the same route repeatedly, you can save that route so that you can load it up again any time you want with this feature.
  • Any route you save will appear at the bottom right of the screen under the "Saved Routes" header. These routes are associated with your account, so they will not disappear when you leave the webiste. However, you must be logged in to use this feature.
  • You will have to name any route you save. We recommend you think of a clever name before saving any route so that it will be easy for you to determine what each saved route corresponds to.
Recommendations
We're pretty seasoned UVA experts here. However, sometimes we miss things. If you are using our "Common Locations" feature, and find yourself asking, "This place has to be here. Where is it?", you can write a recommendation to add a new location to the feature in this tab.
Recommendations List
  • If you've posted a recommendation, this tab is where you will see it. In all likelihood, you will see a series of recommendations here.
  • Each recommendation post has a like button associated with it should you agree with a post. If any post gets enough likes, it is automatically added to the "Common Locations" feature to be used by any user who desires to do so.
  • Each post also has a comments section. If you feel any particular way about a recommendation, feel free to share how you feel.

Now get to exploring!
