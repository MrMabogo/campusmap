If you're new around here, below are a collection of brief summaries detailing what you can expect from the site:
Map Features
Using Location Services
Default Starting Location
IMPORTANT: Of the two search bars found at the top left of the Map tab, the top search bar will have coordinates in it which correspond to your current location ONLY IF you have "Location Services" turned ON. Otherwise, please ignore these coordiantes as they do not yield a location valuable to you.
Current Location Coordinates
Just below the map, you can find your current location coordinates (should your "Location Services" be turned ON). Also, a simple refresh of the page will put your current location coordinates back in the top search bar.
Map Controls
Zoom & Drag
You should be able to drag the map around and zoom in/out at your convenience. IMPORTANT: Should you be trying to drag the map around and it doesn't appear to be moving, BEFORE realeasing the mouse, drag it off to the side away from the map! If you release before doing so, the map may accidentally change your route.
Common Locations Section
Getting Routes to High-Traffic Locations at UVA
When in the "Common Locations" section, you will see a series of dropdown menus. Clicking any of the links a given dropdown will display a route from your current location to the location described by the link.
IMPORTANT: Since this feature makes use of your current location when displaying routes, please ensure your "Location Services" are ON if you plan to use it. Otherwise, the routes displayed by this feature will not be useful for you.
Saving Routes
Saving and Naming
If you find yourself searching for the same route repeatedly, this feature allows you to save said route so that you can load it up again any time you want.
You will have to name every route you save. We recommend you think of a clear/obvious name before saving a route so that it will be easy for you to determine what route each saved route corresponds to.
Saved Routes Section
Any route you save will appear at the bottom right of the screen under the "Saved Routes" header. These routes are associated with your account, so they will not disappear when you leave the webiste. However, you must be logged in to use this feature.
Leaving a Recommendation
Recommended Additions to Common Locations Section
We're pretty seasoned UVA experts here; however, sometimes we miss things. If you are searching through our "Common Locations" feature, and find yourself asking, "This place has to be here. Where is it?", you can write a recommendation to add said new location in the Recommendations tab.
Recommendations Forum
If you've posted a recommendation, the Recommendations List tab is where you will see it. In all likelihood, you will see a series of recommendations here.
Each post has a like button associated with it. If any post gets enough likes (people who agree with it), it is automatically added to the "Common Locations" feature to be used by anybody who desires to do so.
Each post also has a comments section. If you feel any particular way about a recommendation, this section allows you to share how you feel.
